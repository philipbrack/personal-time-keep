WELCOME

This simple app allows you to keep track of how much time you spend on daily tasks

You can manage several concurrent tasks across several projects

If you mess up... No worries! Edit a task to clarify the work periods

This app will always be 100% free and will never monitor you!

USER EXPERIENCE NOTES

WHAT!? An explanation of how to interact with an app? This is an engineer created app so I want to give somes notes in case things are not intuitive.

To edit a task slide it to the left, and click on the edit button

To delete / complete a timer swipe to the right

To pause / resume a timer just click on it

The Icon " " is used to add a new timer or project

So Simple!
